PROJECT VOID
INSTALL FOR THE BEST EXPERIENCE
  1. Open this page in Chrome.
  2. Tap the ⋮ menu icon in the top-right corner.
  3. Tap "Add to Home screen" or "Install app".
  4. Tap "Install" to confirm.
  5. Open Project Void from your home screen icon.
If a bar appears at the bottom of the screen offering to install the app, you can tap that instead.
  1. Open this page in Safari (this must be done in Safari, not Chrome).
  2. Tap the Share icon (square with an arrow pointing up) in the toolbar.
  3. Scroll down and tap "Add to Home Screen".
  4. Tap "Add" in the top-right corner.
  5. Open Project Void from your home screen icon.
iOS only allows installing web apps through Safari - it won't work from Chrome or other browsers on iPhone/iPad.
  1. Open this page in Chrome or Edge.
  2. Look for the install icon (a small monitor with a down arrow) on the right side of the address bar.
  3. Click it, then click "Install".
  4. Open Project Void from your Start menu or desktop shortcut.
Don't see the icon? Click the ⋮ menu instead and look for "Install Project Void...".
  1. Safari: open this page, then use the File menu → "Add to Dock".
  2. Chrome: open this page, click the install icon in the address bar, then click "Install".
  3. Open Project Void from your Dock or Applications folder.
"Add to Dock" requires Safari 17 or later. Older versions can use Chrome instead.
